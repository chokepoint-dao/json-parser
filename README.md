# zkJSON
json parser in lurk


# Description
Let's say there's Alice and Bob. Both know a poseidon hash $H$, a path $P$, and a value $V$. Only Alice knows the preimage of $H$ $J$. zkJSON lets Alice prove to Bob that

- $J$ is a valid JSON.
- The value in $J$ under the path $P$ equals $V$.
